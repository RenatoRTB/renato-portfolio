Projeto Receitas

Meu primeiro projeto, onde foi usado os conceitos básicos de html, como lista ordenada e não ordenada. Também foram usados comandos básicos de CSS para estilização de backgrounds e fontes. Posteriormente o projeto foi reestilizado.

Neste projeto foi criada uma Landing Page para um site ficticio de rock. Foi exigida uma melhor estruturação do código html com o uso de containers. surge então um novo conceito no código CSS com a introdução do Flash Box.
